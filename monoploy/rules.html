Rules

  1. Drive your opponents to bankruptcy for victory
  2. Land on properties to buy them
  3. All three properties of a set must be owned to build houses
  4. Houses must be built evenly
  5. Rolling doubles, or rolling two of the same number, allows you to take another turn
  6. Go to jail if three doubles are thrown in a row
  7. Pay rent of a propertie if owned by an opponent

Tips

  1. If in debt, properties can be mortgaged for half of what it is worth
  2. Mortgaged properties can be unmortgaged for full price
  3. Houses can be sold for half of what the house is worth
  4. The more railroads owned the more money is made from rent
  5. Higher value properties take more money to build houses on, better to build on lower value properties
  6. If you are having trouble getting money to build houses, mortage all the properties not part of a completed set
